# 企劃書 — 2×3 雙線拉霸機(Lucky Double)

- 版本:v1.0
- 日期:2026-06-11
- 基底:現有 1×3 機台(RTP 94.9%)的延伸改版

## 1. 玩法總覽

- 盤面:**2 列 × 3 輪**。三個輪帶相同;每一格獨立依權重抽取
  (無實體輪帶相鄰關係,上下兩格互相獨立)。
- 賠付線:**2 條水平 payline** — 上排一條、下排一條。
- 中獎規則:單一條線上三個 symbol 相同(3-of-a-kind)即中獎,
  其他組合不賠。兩條線分別判定、派彩相加。

## 2. 下注與派彩

- 玩家每把下注 `bet`(整把注金,涵蓋兩條線,不另外按線數加倍)。
- 單線中獎派彩 = 該 symbol 倍率 × `bet`。
- 整把派彩 = 上排派彩 + 下排派彩;結算 `balance += payout − bet`。
- 下注驗證與餘額不足規則沿用現有機台(bet ≥ 1 且 ≤ balance)。

## 3. Symbol 與賠率表

| Symbol | Glyph | 輪帶權重 | 單線倍率 |
|---|---|---:|---:|
| Cherry | 🍒 | 8 | 3x |
| Lemon | 🍋 | 6 | 6x |
| Bell | 🔔 | 3 | 12x |
| Star | ⭐ | 2 | 40x |
| Seven | 7️⃣ | 1 | 484x |

總權重 20。單格出現機率 = 權重 / 20。

## 4. 數學指標(精確值,詳見《機率計算.xlsx》)

| 指標 | 精確分數 | 小數 |
|---|---|---:|
| 整把 RTP | **99/100** | 0.990000 |
| 單線 RTP 貢獻 | 99/200 | 0.495000 |
| 單線 hit frequency | 191/2000 | 0.095500 |
| 整把 hit frequency(至少一條線中獎) | 727519/4000000 | 0.18187975 |

每個 symbol 對整把 RTP 的貢獻:

| Symbol | 單線 P(3-of-a-kind) | 整把 RTP 貢獻 |
|---|---|---:|
| 🍒 | (8/20)³ = 8/125 | 0.384 |
| 🍋 | (6/20)³ = 27/1000 | 0.324 |
| 🔔 | (3/20)³ = 27/8000 | 0.081 |
| ⭐ | (2/20)³ = 1/1000 | 0.080 |
| 7️⃣ | (1/20)³ = 1/8000 | 0.121 |

> 設計說明:🍒🍋🔔⭐ 的倍率為原版之半,在兩條線下整把貢獻與原版一致;
> 相較原版多出的 4.1 個百分點 RTP 全數配給 7️⃣ 大獎(0.080 → 0.121),
> 維持「常中小獎 + 稀有大獎」的體驗曲線。

## 5. 技術要求

- RTP 以有理數(`big.Rat`)解析計算為權威;Monte-Carlo 模擬僅作交叉驗證。
- RNG 一律注入:production 用 `crypto/rand` 種子的 ChaCha8,
  測試/模擬用固定 seed 的 PCG,結果必須可重現。
- 金額狀態與轉動核心分離(Engine 無金額狀態、Machine 記帳)沿用現有分層。

## 6. 驗收標準

1. 解析 RTP 精確等於 **99/100**、單線 hit frequency 精確等於 **191/2000**
   (更新 `TestAnalyticRTPExact` 為新分數)。
2. `make test` 全綠。
3. `make verify`(一千萬次固定 seed 模擬)RTP 與 99/100 差距 ≤ 0.005。
4. Web UI 呈現 2×3 盤面,兩條線分別標示中獎結果。
